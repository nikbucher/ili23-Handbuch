== Über dieses Repository

In diesem Repository wird die Online-Version des INTERLIS 2.3 Handbuchs **"Modellieren raumbezogener Daten"** gepflegt. Das Handbuch bietet eine umfassende Einführung in die Modellierung raumbezogener Daten unter Berücksichtigung von UML und INTERLIS.

Die aktuelle Website ist unter https://nikbucher.github.io/ili23-Handbuch/ erreichbar.
